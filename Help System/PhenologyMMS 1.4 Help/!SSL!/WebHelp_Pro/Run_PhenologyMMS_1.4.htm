How To Run PhenologyMMS 1.4

First be sure to set up the program to Run The execution of the scenario set up in PhenologyMMS. for the crop and location you wish to simulate. This is done via the Begin Setup and the Set Inputs screens.

 

To Run or execute the PhenologyMMS program, click on the Run button located on the Begin Setup, Set Inputs, Temperatures or the Growth Stages screens. This will execute the program and open the Output window.

 

Within the Output window, you can see the inputs to the program. The leaf number The amount of leaves produced by a given growth stage or the number of accumulated GDD's. Leaves are generally not added once the plant reaches the first reproductive stage. table and the phenology table This is part of the output from the model run and shows the predicted Date or number of Days from a specific event, for example, 'Planting' for each growth stage. The columns Day of Year, Date, DAP, DAE and DAV provide this information. The predicted number of Growing Degree-Days for each stage after a specific event, e.g., 'Emergence' or 'Vernalization' are provided in the columns GDD AP, GDD AE, and GDD AV. The Number of Leaves predicted for each vegetative stage is also displayed in the table. The number of leaves shown in the reproductive stages is the same as that at the last vegetative stage and shows that no new leaves are added in the reproductive stages. which resulted from the run are also displayed in the Output window.

 

From the Output screen, you have the option to Save the Scenario or setup for this run and to Save the Output from this run.
